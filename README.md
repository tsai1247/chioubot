# 邱P bot

## 簡介
* 讓邱P可以用telegram的專案

## 使用
### 注意
* 此專案使用到 telegram 官方提供的 pre-release 套件，可能會出現未預期錯誤。若有問題再煩請告知，謝謝。

### enviornment
* windows 應該是都能用，linux系統沒試過

### command
1. `git clone https://github.com/tsai1247/chioubot/`
2. `rename .env.example .env`
3. 修改.env中的token
4. `pip3 install -U -r requirements.txt`
6. `python .\main.py`
